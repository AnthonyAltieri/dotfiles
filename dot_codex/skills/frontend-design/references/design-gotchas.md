# Frontend Design Gotchas

Use this file when the design direction starts collapsing into generic patterns.

## Common failure modes

- The design uses a safe, overfamiliar font stack.
- The layout becomes a sequence of nearly identical cards and sections.
- The palette is technically pleasant but emotionally flat.
- The page uses many small animations instead of one or two meaningful motion ideas.
- Mobile looks like a squashed desktop version instead of a deliberate composition.

## Course corrections

- Pick one memorable typographic move and make the rest of the layout support it.
- Introduce hierarchy through scale, density, overlap, or asymmetry instead of more borders.
- Give the background an actual job: mood, depth, rhythm, or spatial separation.
- If the interface feels too safe, push one dimension harder: type, color, composition, or motion.
- If the interface feels too busy, remove decorative layers until the main idea is obvious again.
